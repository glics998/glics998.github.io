BLOCK vs. INLINE BLOCK ELEMENTS

BLOCK:

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). For example p. ,h1, h2, h3, h4, h5, h6, ol, ul, pre.

INLINE BLOCK:

An inline element does not start on a new line and only takes up as much width as necessary. For example img, br, em, a, b, i.
